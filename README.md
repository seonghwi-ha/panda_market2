# 🐼 판다마켓 로그인 / 회원가입 페이지

## 📌 프로젝트 소개

HTML, CSS를 활용하여 로그인 페이지와 회원가입 페이지를 구현한 프로젝트입니다.
사용자 인터페이스(UI)와 기본적인 페이지 이동, input 인터랙션을 중심으로 제작했습니다.

---

## 🌐 배포 링크

👉 https://seonghwi-ha-pandamarket.netlify.app/

---

## 🚀 주요 기능

### ✅ 공통 기능

- "판다마켓" 로고 클릭 시 루트 페이지("/")로 이동
- SNS 아이콘 클릭 시 외부 링크 이동
  - Google → https://www.google.com/
  - Kakao → https://www.kakaocorp.com/page/

- input focus in 시 테두리 색상 변경 (#3692FF)
- input focus out 시 테두리 제거

---

### 🔐 로그인 페이지

- "회원가입" 버튼 클릭 시 `signup.html` 페이지 이동

---

### 📝 회원가입 페이지

- "로그인" 버튼 클릭 시 `login.html` 페이지 이동

---
